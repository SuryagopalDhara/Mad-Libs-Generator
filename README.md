# Mad-Libs-Generator
Problem Statement : A fun program that asks the user to enter random words (without knowing the story), then reveals a hilarious story using their words.  
#Concepts :  
input() — to take user input 
Variables — to store the words 
f-strings — to insert words into the story 
print() — to display the final story

#V1 : Just Printing the (one) story with given word .
